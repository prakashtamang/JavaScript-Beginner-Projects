## Prime Number Checker

A simple JavaScript application that determines whether a given number is prime or non-prime using conditional statements, loops, and basic mathematical logic.

### Features

- Accepts a number entered by the user.

- Validates the input to ensure it is greater than 1.

- Determines whether the number is prime or non-prime.

- Uses an efficient algorithm by checking divisibility only up to the square root of the number.

- Displays the result instantly on the page.
