## Random Number Generator

A Random Number Generator is a simple yet practical JavaScript project that demonstrates core programming concepts. It helps you understand how to handle user input, generate random values within a specified range, validate input, and dynamically update the user interface.

### Features

- Generate a random number within a user-defined range.

- Accept custom minimum and maximum values.

- Updates the interface dynamically to display results.

- Reset the input fields to their default values and clear the displayed result.
