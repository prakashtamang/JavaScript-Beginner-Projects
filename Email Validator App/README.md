## Email Validator App

A simple JavaScript application that validates whether an entered email address is in a valid format using a regular expression (RegExp). The app provides instant feedback to help users enter correctly formatted email addresses before form submission.

### Features

- Accepts an email address as user input.

- Validates the email format using a regular expression (RegExp).

- Displays a success message for valid email addresses.

- Displays an error message for invalid email addresses.

- Prevents form submission if the email format is invalid.

- Resets the form after successful validation.
