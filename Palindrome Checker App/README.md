## Palindrome Checker App

A simple JavaScript application that checks whether a given word, phrase, or number is a palindrome. The app ignores spaces, punctuation, and letter casing to provide accurate results.

### Features

- Accepts a word, phrase, or number as input.

- Removes spaces and special characters before checking.

- Performs a case-insensitive comparison.

- Reverses the normalized input and compares it with the original.

- Displays whether the input is a palindrome or not.
